# 클래스형 컴포넌트와 함수형 컴포넌트의 차이점에 대해서 설명해주세요.

클래스형 컴포넌트는 class 문법을 사용하고 this 키워드를 사용하여 state와 props에 접근할 수 있습니다.
componentDidMount, componentDidUpdate, componentWillUnmount 등의 생명주기 메서드를 사용할 수 있습니다.

함수형 컴포넌트는 function 또는 화살표 함수를 사용하여 선언합니다.
useState, useEffect 등의 Hooks를 사용하여 state와 props에 접근할 수 있습니다.
코드가 간결하고 가독성이 높습니다.
